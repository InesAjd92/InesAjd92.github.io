Contexte & problématique

L'affaire des ballons espions chinois (février 2023) a généré une vague massive de discours conspirationnistes sur les réseaux sociaux américains. Ce mémoire analyse comment ce discours se structure : quels marqueurs rhétoriques ? Quels registres lexicaux ? Comment le sentiment évolue-t-il dans le temps ?

Résultats clés

Identification de 4 clusters lexicaux distincts : militaire/sécuritaire, politique/complot, médiatique/fake, et émotionnel/anxiété.
Pic de sentiment négatif à J+3 après l'abattage du ballon — la rhétorique conspirationniste s'intensifie après l'action gouvernementale, pas avant.
IRaMuTeQ révèle une opposition lexicale structurante entre "gouvernement" / "vérité" et "caché" / "réel" — structure classique du discours conspirationniste.
Le discours conspirationniste emprunte massivement au registre journalistique pour se légitimer ("sources", "preuves", "révélations").

Méthode technique

Corpus : Tweets et posts Reddit collectés via API sur 3 semaines autour de l'événement.

Analyse : Textométrie sous IRaMuTeQ (classifications, AFC), analyse de sentiment sous R avec le package sentimentr, cartographie lexicale.

Cadre théorique : Rhétorique conspirationniste (Sunstein & Vermeule), post-vérité (McIntyre), analyse critique du discours (Fairclough).
